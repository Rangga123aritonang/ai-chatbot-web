# AI Chatbot Web Application

## 📋 Description
A simple AI chatbot web application built using Java (NetBeans) and OpenAI API. The chatbot responds to user queries automatically.

## 🛠️ Technologies
- Java
- NetBeans
- JDK 22
- HTML & CSS

## 🚀 Features
- User input form
- AI response output
- Simple user interface

## ⚙️ How to Run
1. Clone this repository
2. Open in NetBeans
3. Run the project
